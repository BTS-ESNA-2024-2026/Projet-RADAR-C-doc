## Structure

La documentation est découpée en plusieurs sections accessibles:

- **Accueil** : Présentation générale et sommaire.
- **Installation** : Pré requis, configuration et mise en place.
- **Utilisation** : Commandes, exemples et cas d'usage.
- **Fonctionnement** : Architecture interne et principes techniques.
